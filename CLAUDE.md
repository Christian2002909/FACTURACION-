# Sistema de Facturacion — Instrucciones para Claude

## Eficiencia
- Sin preambulos ("Claro!", "Excelente pregunta!") — ir directo al punto
- Sin resumenes al final de las respuestas
- Liderar con la accion o respuesta, no con el razonamiento
- Respuestas cortas y directas; expandir solo cuando la complejidad lo requiera
- Sin comentarios obvios en el codigo
- Sin docstrings en funciones triviales

## Proyecto
- Sistema de facturacion en Python
- Backend: FastAPI
- Automacion web disponible via browser-use
- Usar agentes especializados para tareas complejas

## Stack
- Python 3.11+
- FastAPI + Uvicorn
- SQLAlchemy (ORM)
- Pydantic (validacion)
- browser-use (automatizacion de navegador)
- ReportLab (generacion de PDFs)
