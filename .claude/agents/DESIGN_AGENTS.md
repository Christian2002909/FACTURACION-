# Design Agents - Sistema de Facturación Paraguay

## Agentes Especializados Habilitados

Los siguientes agentes están activos para el diseño y mejora del sistema de facturación:

### 1. **Software Architect**
- Diseño de arquitectura del sistema
- Patrones de diseño
- Decisiones técnicas estratégicas
- Optimización de componentes

### 2. **Frontend Developer**
- Mejora de la interfaz Tkinter
- Diseño de UI/UX
- Validación de formularios
- Componentes interactivos

### 3. **Backend Architect**
- Diseño de APIs FastAPI
- Estructura de base de datos
- Autenticación y autorización
- Optimización de rendimiento

### 4. **Code Reviewer**
- Revisión de código
- Mejoras de mantenibilidad
- Detección de vulnerabilidades
- Buenas prácticas

### 5. **Rapid Prototyper**
- Prototipado rápido
- MVP (Mínimo Producto Viable)
- Iteración ágil

## Cómo Usar los Agentes

Para activar un agente especializado en tareas complejas:

```
/agent <tipo_agente>
```

### Ejemplos de uso:
- `@Software Architect` - Para decisiones de arquitectura
- `@Frontend Developer` - Para mejoras en la GUI
- `@Backend Architect` - Para diseño de APIs
- `@Code Reviewer` - Para revisión de código

## Integración con el Sistema

Los agentes están disponibles para:
- Análisis del código actual
- Diseño de nuevas funcionalidades
- Optimización del sistema
- Resolución de problemas complejos
- Automatización con browser-use
