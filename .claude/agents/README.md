# Agentes de Diseño Habilitados

## Estado: ACTIVADO ✓

Los agentes especializados para diseño del sistema de facturación están habilitados y listos para usar.

### Agentes Disponibles

| Agente | Función |
|--------|---------|
| **Software Architect** | Diseño arquitectónico, patrones, decisiones estratégicas |
| **Frontend Developer** | UI/UX, formularios, componentes Tkinter |
| **Backend Architect** | APIs FastAPI, base de datos, autenticación |
| **Code Reviewer** | Revisión de código, seguridad, mantenibilidad |
| **Rapid Prototyper** | Prototipado ágil, MVP, iteración rápida |

### Cómo Usar

Para invocar un agente especializado en tareas complejas:

```bash
# Ejemplo: Usar Software Architect para decisiones de arquitectura
claude ... @Software Architect
```

### Ejemplos de Uso

**Redesign de GUI:**
```
@Frontend Developer: Rediseña la GUI del sistema de facturación
para mejorar UX y accesibilidad
```

**Optimización de APIs:**
```
@Backend Architect: Optimiza el diseño de las APIs FastAPI
para mejor rendimiento y escalabilidad
```

**Mejoras de Código:**
```
@Code Reviewer: Revisa el código actual e identifica
vulnerabilidades y mejoras
```

### Arquitectura del Sistema

El sistema está organizado en:
- **Backend**: FastAPI + SQLAlchemy + Pydantic
- **Frontend**: Tkinter GUI
- **Datos**: SQLAlchemy ORM
- **Automatización**: browser-use para tareas web
